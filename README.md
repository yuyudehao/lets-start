# 测试

## 查看状态
没有副作用的

```shell
git status
```
## 初始化仓库

```shell
git init 
```


## 将文件添加到暂存区

```shell
git add filename
```


## 生成版本

```shell
git commit -m"版本的描述信息"
```

## git的配置操做

```shell
git config --global user.name yuyu
git config --global user.email .........
```

## 查看版本信息

```shell
git log
```
